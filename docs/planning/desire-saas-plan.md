# 욕망 자극 SaaS 최종 기획서

> 주제: 인간의 보편적 욕망 3가지(미용/돈/육아)를 자극하는 SaaS/모바일 서비스
> 작성: @기획팀장 (Opus) — R3 최종 결정
> 일자: 2026-04-26

---

## 결론 요약 (한 문단)

**MOMFLUENCE의 코어 가설("부모 = 가장 큰 미사용 크리에이터 풀")은 살리되, 키드플루언서 규제 리스크가 사업을 통째로 날릴 수 있는 ★★★★★ 위협이므로 "아이 얼굴/신상 전면 비노출 + 부모 본인 중심 콘텐츠" 방향으로 전면 재설계한다.** 서비스명은 가칭 **PARENTLY (페어런틀리)** — "아이를 보호하면서, 부모의 일상을 콘텐츠·수익·자기관리로 바꿔주는 부모 전용 크리에이터 SaaS". 미용/돈 욕망은 1차 진입(콜드스타트 해소), 육아는 2차 잠금(리텐션)으로 욕망 점화 순서를 뒤집어 비평팀이 지적한 "콜드스타트 vs AI 품질 딜레마"를 우회한다. 6개월 MVP, 5인 팀, 2.5억 예산, 한국 출시 가능, 수익 3중 구조 유지.

## 채택한 방향과 핵심 변경

### 창의팀에서 가져온 것
- **부모 크리에이터 플랫폼**이라는 코어 개념 (미사용 크리에이터 풀 = 가장 큰 블루오션)
- **나노 크리에이터 수익화** (팔로워 500명부터) — 기존 인플루언서 마켓 사각지대 정면 공략
- **수익 3중 구조** — SaaS 구독 + 플랫폼 수수료(메인) + 커머스/B2B
- **욕망 3중 점화 메커니즘** (단, 순서는 재배치 — 아래 참조)
- **5인 팀 / 6개월 MVP / 2~3억** 실행 사양

### 비평팀에서 반영한 것 (어디에 어떻게)
| 비평 지적 | 반영 위치 | 구체 변경 |
|---|---|---|
| 키드플루언서 규제(★★★★★) | **서비스 정책 v0** | 아이 얼굴·실명·생년월일·학교 노출 차단을 시스템 레벨에서 강제. AI 자동 블러/뒷모습 합성. |
| FTC 1.7억 달러 / COPPA / 한국 청소년보호 | **법무 가드레일** | 14세 미만 콘텐츠는 부모 본인 시점만 허용. 가족 콘텐츠는 "부모의 시선" 포맷으로 한정. |
| 콜드스타트 vs AI 품질 딜레마 | **욕망 점화 순서 재배치** | 1차=미용(셀프 메이크업/뷰티) → 2차=돈(첫 협찬) → 3차=육아(부모 커뮤니티 잠금). 미용은 AI 데이터 없이도 즉시 작동. |
| 광고주 의존 → 사용자 신뢰 인질 | **수익 구조 비중 조정** | 1년차 SaaS 구독 비중 40% 이상 강제(구독이 신뢰 앵커). 광고 수수료 비중 60% 상한. |
| 운영 인력 비용 선형 증가 | **MVP 범위 축소** | 1차 출시는 "AI 콘텐츠 제작 + 협찬 매칭" 2개만. 굿즈 커머스는 6개월 차 이후. |
| 대안 B (단일 욕망 집중) | **포지셔닝 메시지** | 마케팅에선 "엄마의 자기관리"(미용+돈)로 좁혀 진입, 육아는 내부 락인 장치로만 사용 |
| 대안 A (전문가 협력형) | **신뢰 인프라** | 협찬 브랜드 검증에 육아·미용 전문가 패널 도입 (Curology 모델 차용) |
| 키드플루언서 착취 프레임 | **PR 메시지** | "아이를 지키면서 부모가 빛나는 플랫폼"을 전면 슬로건으로 |

### 새로 추가한 것 (제3의 안)
- **"보호 모드(Protect Mode)" 기본값 ON** — 아이 얼굴 자동 블러는 옵션이 아니라 디폴트. 끄려면 다중 동의 + 워터마크.
- **"부모 매너온도"** (대안 B의 당근마켓식 신뢰 인프라 차용) — 협찬 이행률, 콘텐츠 품질, 커뮤니티 평판 통합 점수. 광고주가 이 점수로 매칭.
- **욕망 점화 순서 역전** — 창의팀 안(육아→미용→돈)을 (미용→돈→육아)로 뒤집음. 미용은 AI 콜드스타트 면역, 돈은 즉시 보상, 육아는 가장 강한 락인이라 마지막에.

## 서비스 개요

- **이름(가칭)**: PARENTLY (페어런틀리)
- **한 줄 정의**: "아이를 노출하지 않고도, 부모의 일상을 콘텐츠와 수익으로 바꿔주는 부모 전용 크리에이터 SaaS"
- **타깃**: 25~40세 한국 여성 부모 (1차) / 부모 예정자·신혼 (2차). 인스타·블로그 사용 경험 있고 팔로워 100~5,000명대인 일반인.
- **핵심 가치**: "아이는 안전하게, 나는 빛나게, 통장은 채워지게"

## 욕망 자극 메커니즘 (재배치된 순서)

| 단계 | 욕망 | 트리거 | 첫 도파민 | 락인 효과 |
|---|---|---|---|---|
| 1차 (가입 후 24시간) | **미용** | 셀카 → AI 메이크업/스타일 분석 + "당신과 닮은 부모 인플루언서 TOP 3" | "나도 저 톤이 어울리네" 시각적 보상 | 콘텐츠 1개 자동 생성 |
| 2차 (가입 후 7일) | **돈** | 매너온도 50점 도달 → 첫 나노 협찬 매칭 ("3만원 뷰티템 1개") | 첫 수익 알림 | 협찬 이행 → 매너온도 상승 |
| 3차 (가입 후 30일) | **육아** | 비공개 부모 커뮤니티 잠금 해제 (매너온도 기반) | "나만 아는 진짜 육아 정보" 소속감 | 끊을 수 없는 커뮤니티 의존 |

**왜 순서를 뒤집었는가**: 비평팀이 지적한 콜드스타트 문제는 "초기 AI가 범용과 다를 게 없다"는 것. 미용은 셀카 1장으로 즉시 시각 결과를 줄 수 있어 데이터 없이도 작동한다. 육아는 가장 데이터·시간 의존적이라 마지막 단계로.

## 핵심 기능 5개

1. **AI 미용 디아그노시스 + 부모 인플루언서 매칭** (1차 욕망)
   - 셀카 1장 → 톤·스타일·연령대 분석 → 닮은 부모 크리에이터 노출 → 클릭 한 번에 유사 콘텐츠 생성
2. **나노 협찬 매칭 엔진** (2차 욕망)
   - 매너온도 ≥ 50점 부모에게 3~10만원대 마이크로 협찬 자동 매칭. 광고주는 PARENTLY가 사전 검증.
3. **보호 모드 콘텐츠 빌더**
   - 사진/영상 업로드 시 아이 얼굴/이름/학교 자동 감지·블러·익명화. 디폴트 ON. 끄기 어렵게 설계.
4. **부모 매너온도 + 비공개 커뮤니티** (3차 욕망 락인)
   - 협찬 이행률 + 콘텐츠 품질 + 커뮤니티 평판. 70점 이상만 진입 가능한 "리얼 부모 라운지".
5. **SaaS 대시보드** (수익화 백본)
   - 콘텐츠 성과·수익·매너온도·다음 협찬 추천을 한 화면에. 무료/Pro(월 9,900원)/Pro+(월 29,000원).

## 차별화 포인트

1. **"아이 비노출"을 디폴트로 강제하는 유일한 부모 플랫폼** — 경쟁사(인스타, 틱톡, 블로그)는 못 막는다. 우리는 시스템 레벨로 막는다. 이게 곧 PR 자산이고 규제 방패.
2. **나노 협찬(3~10만원대) 자동 매칭** — 기존 인플루언서 에이전시는 1만 팔로워 미만은 안 받는다. 우리는 500명부터.
3. **부모 매너온도** — 광고주 입장에서 "이 부모는 진짜 협찬 잘 이행하는가"의 신뢰 인프라. 당근마켓이 중고거래에서 한 일을 부모 협찬 시장에서 한다.
4. **욕망 3중 점화 + 락인** — 미용으로 데려와서, 돈으로 묶고, 육아 커뮤니티로 못 떠나게.

## PM이 결정해야 할 사항 (질문 3개 + 현재 가정)

> 비평기획자가 던진 3개 질문에 대해, PM이 답을 안 줬으므로 팀장이 임시 가정을 세움. **다른 답을 원하시면 기획이 바뀝니다.**

1. **광고주 이익 vs 사용자 이익 충돌 시?**
   - **현재 가정**: **사용자(부모) 우선**. 매너온도 양방향, 광고주도 평가받음. 단기 매출 손해 감수.
2. **감수 가능한 최대 법적 리스크?**
   - **현재 가정**: **중하 수준**. 의료·유전자 영역은 진입 안 함. 키드플루언서 영역은 보호 모드 강제로 우회. 광고 표시(#광고) 자동화는 출시 1일차부터 의무화.
3. **3가지 욕망 동시 자극 = 획득용 vs 리텐션용?**
   - **현재 가정**: **미용=획득, 돈=전환, 육아=리텐션**. 동시가 아니라 시간차 점화.

## 보류/추가 검토 사항

- **글로벌 확장 시점** — 1년차는 한국 단독. 일본/대만 확장은 1년 후 데이터 보고 결정.
- **남성 부모(아빠) 타깃** — 1차 MVP는 엄마 타깃. 아빠는 6개월 데이터 후 별도 검토.
- **AI 콘텐츠 표시 의무** — EU AI Act 영향. 한국도 곧 따라올 가능성. 지금부터 메타데이터 워터마크 설계 권고.
- **창의팀의 SKINDNA(유전자 미용) 흡수안** — 비평팀이 지적한 23andMe 데이터 유출 선례 고려해 **1년차 보류**. 자체 유전자 데이터 수집은 안 함.
