# "하루 10만 방문 + 광고수익 + 매일 확인" 웹사이트 도메인(주제) 발굴 보고서

## 0. 전제 교정 — 반드시 먼저 알아야 할 두 가지

### (1) "하루 10만 방문"은 목표가 아니라 복권이다
비평 검증에 올린 후보 8개 **전원 verdict "unrealistic"**. 실측 근거:
- 국내 퀴즈웹 1위 machugi.io 전체가 일 2.9만 방문 (Semrush)
- 참여형 투표 1위 피쿠(piku)가 9년 운영에 일 3.3만
- 꼬맨틀(2022 신드롬의 주인공)조차 현재 일 ~3,100 방문, 월 -29% 하락 (Similarweb)
- Wordle 본가도 7개월 만에 공유 91% 급락 — 데일리 포맷은 감쇠가 기본값
- 신규 사이트가 한국에서 오디언스·예산 없이 DAU 10만에 도달한 사례: **검색으로 확인되지 않음**

→ **사업 판단 기준선은 DAU 5,000~10,000으로 재설정**하고, 10만은 여러 사이트 포트폴리오 합산의 장기 상한으로만 취급할 것.

### (2) 현 배포 모델로는 광고수익이 0원이다
- 애드센스는 **소유한 최상위 도메인**을 요구 — `project-orsrw.vercel.app/<slug>` 공유 서브도메인+경로로는 신청 자체 불가
- 승인에는 통상 1,500자+ 글 20~30개 + 소개/개인정보처리방침/문의 페이지 필요 — 게임·도구 단일 페이지는 구조적으로 거절됨 ("가치 있는 콘텐츠 부족")
- 게임 플레이 화면 자체에는 광고 게재 금지 정책 (게시자 콘텐츠 없는 화면)
- 심사 리드타임 2주~1개월+ → 런칭 후 1~2개월은 수익 0으로 계상
- 대안으로 거론되는 카카오 애드핏은 실측 eCPM 26~90원 — 애드센스(RPM 1,000~5,000원)의 1/10 이하, 대안 불가

→ 이 프로젝트는 예외적으로 **자체 도메인 구매 + 독립 Vercel 프로젝트**로 가야 하며, "어제의 정답 해설" 같은 텍스트 콘텐츠 층을 게임 루프 안에 설계해야 한다.

### 광고수익 현실 (한국 트래픽 실측 RPM 기반)

| DAU | 월 PV (PV 2~3/인) | 월 수익 (RPM 1,000~2,500원, 도구형 하단값) |
|---|---|---|
| 3,000~8,000 (6개월 현실 기대) | 18만~72만 | **월 20만~150만원** |
| 10,000 (목표 기준선) | 45만~90만 | 월 45만~270만원 |
| 100,000 (원 목표 — 전례 없음) | 450만~900만 | 월 450만~1,500만원 |

## 1. 최종 랭킹 TOP 3

세 후보는 경쟁 관계가 아니라 **하나의 "데일리 게임 허브" 전략**으로 수렴한다. 1위를 1번 타자로 출시하고, 반응이 있으면 같은 도메인에 2·3위를 모드로 증설해 재방문 표면과 1인당 PV를 늘린다 (Worldle이 Teuteuf 번들로 생존한 방식).

### 🥇 1위 — "오늘의 동네" (한국판 데일리 지리게임)
매일 시·군·구 하나의 실루엣이 출제되고, 6번 안에 맞추는 게임. 틀릴 때마다 거리(km)·방향 화살표·근접도% 힌트. Worldle(일 18만, 전세계) 포맷의 한국 행정구역판.

**왜 이게 이기는가:**
- **법적으로 유일하게 깨끗함** — 8개 후보 중 7개가 퍼블리시티권·저작권·트레이드드레스 리스크 보유. 행정구역 경계는 국토부/통계청 공공데이터(GeoJSON)로 리스크 사실상 0
- **운영비 0에 수렴** — 시·군·구 250개 사전 큐잉으로 8개월 무인 운영. 1인+AI팀 체제에 맞는 유일한 구조
- **광고 승인과 정합** — '어제의 동네' 해설 페이지(지역 정보·역사·특산물)가 매일 쌓이며 애드센스 요구 콘텐츠 층 + SEO 롱테일("○○구 인구·특산물")을 자동 생산
- **한국형 전파 훅** — "오늘 정답이 우리 동네" 뱃지, 지역별 정답률 리더보드(지역 대항전) → 에펨코리아식 지역 밈, 카톡 단톡방 대항전
- **빌드 1~2일 최단** (+전파 0.5일 +광고기반 1일 = 총 3일)

**현실적 기대치:** 6개월 시점 안정 DAU 2,000~8,000. 시딩이 밈으로 꽂히는 최선 시나리오 상한 1~2만.

**MVP 스코프 (3일):**
- D0: 자체 도메인 구매·연결. 작명에서 '-들(-dle)' 회피(NYT 트레이드드레스), 🟩🟨 그리드 공유 금지(거리·방향·근접도만)
- 코어(1~2일): GeoJSON→실루엣 SVG, haversine 거리·방향, 지명 자동완성, 6회 시도, localStorage 스트릭, 250개 출제 큐
- 전파(0.5일): 스포일러-프리 카톡 OG 카드(오후의 패 OG 스택 재사용), 단톡방 대항전 코드
- 광고 기반(0.5~1일): '어제의 동네' 해설 템플릿+아카이브, 소개·개인정보처리방침·문의, 애드센스 신청
- **명시적 제외:** 로드뷰 모드 — 카카오/네이버 지도 API는 게임 사용 금지 공식 확인(설계에서 영구 삭제), 로그인, 앱

**킬 기준:** 30일 내 DAU 500 미만 정착 시 즉시 손절 (3일치 비용만 잃는 실험으로 설계).

### 🥈 2위 — "얼마게" (다이소 한정 오늘의 가격 맞추기)
매일 다이소 상품 사진 1장, 가격 맞추기. 다이소 균일가 구조라 유일하게 '전국 단일 정답'이 성립. 미국 Costcodle(일 ~4,700 방문) 포맷의 한국판.
- 강점: **쿠팡파트너스 커머스 제휴로 애드센스 의존 탈피 가능** — 수익 상방은 전 후보 중 최고
- 약점: 매주 상품 선정·촬영이라는 상시 운영 노동. 허브 2호로 증설이 적합

### 🥉 3위 — "케이들" 안전 풀 버전 (역사인물·지명·드라마 제목 추리)
훅 강도와 현실 DAU 상단(2천~1만)은 전 후보 중 최고였으나, **연예인 풀은 퍼블리시티권 성문화(부정경쟁방지법 타목)로 법률 검토 전 착수 금지**. 안전 풀(역사인물·지명 등) 버전으로 허브 3호 편입이 합리적.

### 탈락 후보 (비평 요지)
- 데일리 밸런스 투표: 피쿠(9년, 일 3.3만)의 벽 — 6개월 500~5,000
- 데일리 한글 퍼즐 허브: 카테고리 천장이 꼬맨틀·꼬들 각 일 ~3,000 — 신규 수요 없음
- K-헤들(K팝 추측): Heardle 감쇠 -85%/8개월 + 저작권·퍼블리시티 리스크
- 전국 티어 결정전: 주제 교체마다 재점화 필요한 소모성 바이럴
- 무가입 3초 운세: 동일 모델 생존자 실측 일 140~400명 — SEO 신규 도메인 열세

## 2. 유입 로드맵 (0 → 1만 → 10만)

- **0→1,000 (D0~D30):** 디시·에펨코리아 지역 게시판·오픈채팅 시딩, 기존 라이브 자산(office-archetype, /pae) 크로스 배너, 단톡방 대항전 훅. 30일 킬 기준 적용
- **1,000→10,000 (M2~M6, 실질 승부처):** '어제의 동네' 250개 SEO 페이지 축적 + 지역 대항전 밈화 + 허브 2·3호 증설(상호 트래픽 급유) + 웹푸시·홈화면 추가
- **10,000→100,000 (조건부, 계획 아님):** 오가닉 도달 전례 없음. 가능 경로는 (a) 유튜버·스트리머 방송 소재화(피쿠의 제2 전성기 경로) (b) 허브 5종+ 포트폴리오 합산 (c) 플랫폼 제휴뿐

## 3. 정면 리스크

1. 10만 미달이 기본 시나리오 — 이 승인은 "10만짜리 사업"이 아니라 "3일짜리 실험" 승인
2. 포맷 감쇠 기본값 — 시딩 스파이크의 5~10%만 잔존 가정
3. 콘텐츠 소진 8개월 — 시즌2는 읍면동·랜드마크·지형지물(공공누리 사진)로
4. 해자 없음 — machugi.io가 1주 내 복제 가능. 방어는 속도와 초반 90일 커뮤니티 선점뿐
5. 수익화 리드타임 — 승인까지 1~2개월 수익 0
6. 단일 실패점 = 커뮤니티 시딩 (신규 도메인 SEO는 6~12개월 잠김)

## 4. 팀장 최종 의견

"오늘의 동네"를 30일 킬 기준부로 착수 승인 요청. 단, **최소 리스크 데일리 트래픽 실험 + 허브 전략 1번 타자**로 승인할 것. 원 목표(하루 10만 + 월 천만원급 광고수익)를 유지하려면 이 카테고리가 아니라 다른 주제 발굴 라운드를 다시 도는 것이 정직한 답이다.
