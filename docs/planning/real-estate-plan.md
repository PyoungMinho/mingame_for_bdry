# 진짜집 (가칭) 기획 최종안

> **한 줄 정의**: "실거주 세입자가 찍은 사진과 다중 검증 신뢰 스코어로, 허위매물을 '걸러서 보여주는' 부동산 중개 플랫폼"

작성: 기획팀장 (ROUND 3 최종 결정) · 2026-06-03
기준 문서: 이 문서는 다음 단계(팀장 회의 → 디자인 → 개발 → QA)의 기준이 된다.

---

## 결론 요약

세입자 사진은 **'필수 조건'이 아니라 '신뢰의 차별적 배지'**로 간다. 핵심 차별점(세입자가 찍은 진짜 사진)은 끝까지 지키되, 그것을 매물 등록의 *진입 장벽*으로 쓰지 않고 *상단 노출 우대의 보상*으로 쓴다. 이렇게 하면 비평팀이 지적한 S급 리스크(공실 30~40% 원천 배제로 인한 양면시장 콜드스타트 악순환)를 회피하면서, 창의팀의 야심(살아있는 매물 인증, 다중 신뢰 스코어)을 살린다. 초기 타깃은 **소비자(세입자) 우선**, 지역은 **서울 관악(신림)·마포(신촌) 1인가구 원룸/오피스텔에 극단적으로 집중**한다. 위험한 아이디어(스테이징 임차인 = 회당 20~50만 원 + 연출 논란으로 자기부정)는 **전면 폐기**, 금전 리워드는 어뷰징 유발 우려로 **'이사 확인 인증서'(임대차 이력 증명)로 변형**한다.

## 1. 문제 정의 — 왜 지금인가

### 직방·다방의 구조적 한계
- **등록 주체와 검증 주체가 동일**: 매물을 올리는 중개사가 사진·정보까지 통제한다. "더 좋은 매물 미끼 → 방문 유도 → 다른 집 권유"가 수익 구조와 맞물려 있어, 자정이 구조적으로 어렵다.
- **이미 거래된 매물이 계속 노출**: 공동중개 구조상 팔린 물건이 미끼로 남는다.
- **데이터로 확인된 심각성**: 한국인터넷광고재단·입법정책연구원 조사에서 매물 200건 중 **91건(약 45%)이 허위·과장**으로 나타났다. 2025년 대학가 모니터링에서도 1,100건 중 321건(약 29%)이 위법 의심.

### 왜 지금인가 (타이밍)
- 정부가 **부동산 매물 DB와 실거래·임대차신고 DB를 연계**해 거래완료 여부를 모니터링하기 시작했고, 허위·과장 광고에 **최대 500만 원 과태료**를 부과한다. 규제 환경이 "진짜 매물"에 유리하게 기울었다 — 우리 콘셉트가 규제와 같은 방향이다.
- 법은 채찍을 들었는데도 청년 대상 허위매물이 여전하다는 보도가 이어진다 → **규제만으로 안 풀리는 빈틈 = 우리의 시장.**

## 2. 타깃 사용자

| 구분 | 대상 | 핵심 니즈 |
|---|---|---|
| **1차 (수요·소비자)** | 서울 관악·마포의 20~30대 1인가구, 대학생·사회초년생 세입자 | "발품 팔기 전에 진짜인지 알고 싶다", 허위매물 피해 회피 |
| **1차 (공급·콘텐츠)** | 퇴거 예정/현 거주 세입자 | "내 방을 인증하면 임대차 이력 증명·다음 이사 혜택" |
| **2차 (B2B)** | 지역 공인중개사 | "진짜 매물 표시로 문의 전환율↑, 신뢰 브랜딩" |

> **핵심 통찰**: 세입자는 '콘텐츠 공급자'이자 '미래의 소비자'다. 이 이중성이 콜드스타트의 열쇠다.

## 3. 핵심 해결 메커니즘

### (A) 매물 등록은 기존 방식 유지, 사진 신뢰도만 차별화
- **등록**: 집주인이 중개사를 통해 올린다(기존과 동일 — 공급 마찰 최소화).
- **사진 등급 3단계** (강제 아님, 표시만):
  - **🟢 실거주 인증 (Gold)**: 현/직전 세입자가 본인 업로드 + EXIF 메타데이터(GPS·타임스탬프) 정합 + AI 생활흔적 스코어 통과
  - **🟡 현장 인증 (Silver)**: 공실 등으로 세입자 사진이 불가할 때, 중개사가 **앱 내 카메라로만** 촬영(외부 업로드 차단) → GPS·타임스탬프 강제 기록
  - **⚪ 미인증 (Unverified)**: 외부 사진·출처 불명 → 노출은 되나 하단 + 경고 태그

### (B) 신뢰 스코어 (매물 카드에 수치 공개)
가중치는 **단일 인증 의존을 피하는** 다중 검증 구조 (비평팀이 "구조적으로 옳다"고 인정한 부분):

| 항목 | 배점 | 검증 방법 |
|---|---|---|
| 실거주 세입자 사진 | 40 | 본인인증 업로드 + AI 생활흔적 |
| 등기부 소유자 본인인증 | 20 | 집주인 동의 기반 등기 연계 |
| 사진 EXIF 메타데이터 정합 | 15 | 촬영 위치·시각 일치 |
| 커뮤니티 신고 이력 없음 | 15 | 신고 누적 시 차감 |
| **거래완료 DB 정합 (실거래·임대차신고)** | 10 | 정부 연계 DB로 "이미 나간 매물" 자동 강등 |

- **80+ = Gold / 60~79 = Silver / 미만 = Unverified.**
- 검색 결과는 **기본값이 '신뢰순(스코어 내림차순)'** — 강제 없이 진짜 매물이 위로 올라온다.
- **AI 자동 블러링**: 업로드 전 인물 얼굴·개인 소지품(택배 송장, 서류 등) 자동 검출·블러 → 게시 전 단계에서 법적 리스크 차단.

## 4. 3대 의사결정 결론

### ✅ DP-1: 세입자 사진 = '필수 조건' vs '차별적 배지' → **차별적 배지로 결정**

- **결론**: 세입자 사진은 매물 등록의 필수 조건이 아니라, **신뢰 스코어 +40점 + 상단 노출이라는 강력한 보상**으로 작동하는 차별적 배지로 둔다.
- **근거**:
  1. **공급 절벽 회피**: '필수'로 하면 공실 30~40%가 첫날부터 배제된다. 양면시장에서 공급이 얇으면 소비자가 안 오고, 소비자가 없으면 세입자가 사진을 올릴 이유도 없다 — 콜드스타트 악순환이 *확정*된다. 비평팀의 S급 ①번 지적을 정면으로 수용.
  2. **콘셉트는 안 희석된다**: 배지로 두되 **기본 정렬을 신뢰순**으로 하면, 화면 상단은 사실상 세입자 인증 매물이 점령한다. "세입자 사진 = 좋은 매물"이라는 사용자 학습이 자연 발생 → 차별점은 *경험*으로 전달된다.
  3. 창의팀의 이원화 트랙(Gold/Silver)을 흡수하되, '필수냐 배지냐'라는 핵심 결단은 배지 쪽으로 명확히 내렸다 — 어정쩡한 절충이 아니라, **"노출은 열되 신뢰로 줄 세운다"**는 한 방향 원칙이다.

### ✅ DP-2: 초기 타깃 = 소비자(세입자) vs 중개사(B2B) → **소비자(세입자) 우선으로 결정**

- **결론**: 6~9개월간 **소비자 신뢰 브랜딩에 올인**. 중개사 수익화(SaaS 구독)는 신뢰 자산이 쌓인 뒤 후행한다.
- **근거**:
  1. **콘셉트와 이해관계 정렬 문제**: 중개사 우선으로 가면, 핵심 콘셉트(중개사가 못 꾸미게 막는 것)가 *돈 내는 고객(중개사)의 이해관계*에 종속된다. 비평팀 지적대로 자기모순 — 우리 제품의 존재 이유를 스스로 무너뜨린다.
  2. **신뢰가 곧 해자**: 우리의 유일한 차별점은 "여기 매물은 믿을 만하다"는 소비자 인식이다. 이건 B2B 영업으로 못 산다. 소비자 신뢰 → 트래픽 → 그때 중개사가 "여긴 문의가 온다"며 *제 발로* 들어오게 만드는 순서가 맞다.
  3. **둘 다 잡으려다 둘 다 잃는다**는 비평팀 경고 수용. 단, "중개사를 적으로 돌리지 않는다" — 보이콧 방어책은 5번·8번에서 별도 설계.

### ✅ DP-3: 론칭 범위 — 얼마나 좁힐 것인가 → **극단적 집중(서울 관악·마포 / 원룸·오피스텔만)으로 결정**

- **결론**: 출시 시점에 **서울 관악구(신림)·마포구(신촌) 2개 권역, 원룸·오피스텔 단일 유형**만 다룬다. 아파트·전세·타 지역 전면 제외.
- **근거**:
  1. **공급 밀도가 양면시장의 생명**: 전국·전유형은 매물 밀도가 분산돼 "검색하면 몇 개 안 나옴 → 이탈"이 된다. 한 동네에서 "여긴 매물이 빽빽하고 다 진짜네"를 만들어야 한다.
  2. **타깃 적합성**: 해당 권역은 이사 주기가 짧고(세입자 사진 회전 빠름), 앱·리워드 친화적인 청년층이며, 허위매물 피해 빈도가 높다(2025년 대학가 모니터링도 이 패턴 확인).
  3. **에브리타임 등 대학 커뮤니티 제휴**로 퇴거 예정 세입자 사진 캠페인을 *핀셋*으로 꽂을 수 있다 — 전국 마케팅 대비 CAC가 압도적으로 낮다.

## 5. MVP 범위

### 이번에 만드는 것 (핵심 4)
1. **매물 검색·상세 (신뢰순 정렬 기본)** — 신뢰 스코어 배지가 매물 카드에 수치로 노출.
2. **세입자 사진 인증 업로드** — 본인인증 → AI 자동 블러링 → EXIF 검증 → 생활흔적 스코어링 → 등급 자동 부여.
3. **신뢰 스코어 엔진** — 5개 항목 가중 합산 + 정부 거래완료 DB 연계로 자동 강등.
4. **커뮤니티 신고** — 사용자가 의심 매물 신고 → 스코어 차감 + 누적 시 자동 검수 큐.

### 이번엔 안 할 것 (명시적 제외)
- ❌ **스테이징(연출) 임차인 고용** — 회당 20~50만 원 비용 + "연출 사진" 논란으로 핵심 차별점 자기부정. **영구 폐기.**
- ❌ **금전 포인트 리워드(현금성)** — 가짜 인증 산업화 우려. 대신 '이사 확인 인증서' 제공(아래 9번).
- ❌ **건물주 직접 사진 허용** — 세입자 전용 콘셉트 희석. 공실은 'Silver(중개사 앱내 촬영)'로만 처리.
- ❌ **아파트·전세·매매·전국 확장** — 콜드스타트 검증 전까지 보류.
- ❌ **IoT 센서 기반 실거주 검증** — 방향은 옳으나 1~2년 데이터 축적 전제, MVP 무기로 부적합(비평팀 지적 수용). 로드맵 후순위.
- ❌ **B2B SaaS 구독 강제** — 6개월 무료 운영, 유료화는 PMF 후.

## 6. 수익 모델

> **원칙**: 출시 6~9개월은 *수익화하지 않는다.* 신뢰·트래픽 자산을 먼저 쌓는다. 이후 단계적 도입.

1. **(후행) 인증 유지 SaaS — 중개사 대상**: Gold/Silver 인증 운영·우선노출 관리 도구. **초기 6개월 완전 무료**(보이콧 완화), 이후 월 29,000원. 전환 설득 근거로 "Gold 매물 문의 전환 데이터"를 대시보드로 공개.
2. **(후행) 성과형 리드 광고**: Gold 매물 문의 1건당 리드 수수료. 신뢰 매물에만 붙여 "광고=양질" 인식 유지.
3. **(탐색) 이사 부가서비스 제휴**: 청소·이사·보험 등 — 세입자 라이프사이클 데이터 기반. PMF 후 검토.

> B2B 촬영 상품(중개사에게 사진을 파는 모델)은 **자기모순이므로 채택 안 함**(비평팀 지적 수용).

## 7. 콜드스타트 전략

**핵심 순서: 공급(세입자 콘텐츠) 먼저 → 일정 밀도 확보 → 수요(소비자) 마케팅.**

1. **0주차 — 지역 락인**: 관악(신림)·마포(신촌) 2개 권역만. 지역 공인중개사 지부에 **6개월 무료 Gold 인증** 제안으로 매물 베이스 확보(등록 마찰 0).
2. **공급 캠페인 (세입자 사진)**: 에브리타임·대학 커뮤니티 제휴 → "퇴거 예정자 사진 인증 캠페인". 보상은 현금이 아니라 **'이사 확인 인증서'(다음 임대차 시 활용 가능한 거주 이력 증명)** + 다음 이사 시 제휴 할인.
3. **밀도 게이트**: 권역별 **인증 매물 50건 + 전체 매물 200건** 확보 전까지 소비자 마케팅 보류(돈 안 쓴다).
4. **수요 점화**: 밀도 충족 후 "신림에서 진짜 방만 보기" 캠페인 — 지역 타깃 광고 + 커뮤니티 입소문.
5. **자동 위임(법적 안전 버전)**: 퇴거 1개월 전 알림 → 세입자 *명시적 동의 시에만* 사진 보존. 이사 완료 시 촬영 권한 자동 만료. (창의팀의 자동 위임 아이디어를 동의 기반으로 안전화.)

## 8. 리스크와 대응

### 법적 리스크 (S급)
- **초상권·개인정보**: 한국 판례상 **촬영·공표 동의의 입증 책임은 게시자(우리)에게 있다.** → ① 업로드 전 **AI 자동 블러링(얼굴·소지품·서류) 필수 통과**, ② 세입자 본인 업로드 + **전자 동의서**(촬영·게시 주체가 세입자임 명시), ③ 약관에 **게시 주체 = 세입자, 플랫폼은 매개**임을 명확히(면책 구조), ④ 신고 시 즉시 내림(notice-and-takedown).
- **규제 정렬**: 정부 거래완료 DB 연계를 *우리 스코어에 내장* → 규제와 충돌이 아니라 **규제를 제품 기능으로 흡수.** (오히려 규제 강화가 우리에게 유리.)

### 어뷰징 리스크 (S급)
- **가짜 인증 산업화 차단**: 현금 리워드 제거(→ 인증서로 대체)로 금전 동기 원천 제거. EXIF·GPS·생활흔적 AI 다중 검증으로 "남의 집 사진 갖다 붙이기" 차단. **허위 인증 적발 시 중개사·계정 즉시 정지.**

### 중개사 보이콧 리스크 (A급)
- **적이 아니라 도구로**: 등록 방식은 기존 그대로(추가 부담 0) + 6개월 무료 + "Gold 매물이 문의가 더 온다"는 데이터로 협력 유인.
- **노출권 보복 방어**: 강제(필수)가 아니라 *우대(배지)* 구조라, 중개사가 "안 하면 손해"를 스스로 느끼게 설계 → 일방적 보이콧 동기 약화. (다방으로 이동해도, 우리만의 신뢰 브랜딩은 소비자 쪽에 있음.)

## 9. '이사 확인 인증서' — 금전 리워드의 대체재 (설계 포인트)

- 세입자가 거주 사진을 인증하고 계약/거주가 확인되면, **임대차 거주 이력을 증명하는 디지털 인증서** 발급.
- **가치**: 다음 집을 구할 때 "성실 거주자" 신뢰 자산으로 활용(보증금·신원 신뢰에 도움). 현금이 아니므로 **가짜로 만들 동기가 낮다**(비평팀 대안 수용).

## 10. 마일스톤 (0 → 1)

| 단계 | 기간(가늠) | 목표 | 완료 기준 |
|---|---|---|---|
| **M0. 기반** | ~6주 | 검색/상세 + 인증 업로드 + 스코어 엔진 + 블러링 MVP | 관악·마포 매물 등록 가능, 스코어 자동 산출 |
| **M1. 공급 채우기** | ~8주 | 중개사 무료 온보딩 + 세입자 사진 캠페인 | 권역별 인증 50건 / 전체 200건 |
| **M2. 수요 점화** | ~6주 | 소비자 마케팅 시작(밀도 게이트 통과 후) | 주간 활성 검색자(WAU) 확보, 문의 발생 |
| **M3. 루프 검증** | ~8주 | 세입자→소비자→재인증 선순환 측정 | 인증 매물 문의 전환율 vs 미인증 유의차 확인 |
| **M4. 수익화 실험** | M3 이후 | SaaS 유료 전환 + 리드 광고 베타 | 유료 전환 의향/실제 결제 검증 |

> 권장: 각 STEP 완료 시 PM/사장(사용자)에게 산출물 보고 후 다음 단계 진행.

## 11. 성공 지표 (KPI)

1. **인증 매물 비율 & 밀도** (북극성 후보): 타깃 권역 내 *Gold/Silver 인증 매물 수*와 전체 대비 비율. 양면시장 공급 건강성의 직접 지표.
2. **인증 매물 문의 전환율 vs 미인증** (가치 증명): "세입자 사진이 실제로 문의를 더 만드는가" — 차별점이 작동하는지의 핵심 증거이자, 중개사 유료화 설득 근거.
3. **허위매물 신고율 / 신고 후 처리 시간** (신뢰 지표): 우리 플랫폼의 허위매물 신고율이 업계(직방·다방) 대비 낮게 유지되는가 — 브랜드 약속의 정량 증명.

## 보류 / 추가 검토 사항

- **정부 거래완료 DB·등기부 연계의 실제 API 가용성·제휴 조건** — 백엔드팀 회의에서 기술/법무 검증 필요(스코어 10+20점이 여기 의존).
- **AI 생활흔적 스코어링·자동 블러링의 정확도 임계치** — 오탐(진짜를 가짜로) 시 공급자 이탈, 미탐(개인정보 노출) 시 법적 리스크. QA 단계 핵심 검증 항목.
- **'이사 확인 인증서'의 실효성** — 다음 임대차에서 실제로 신뢰 자산으로 통용될지 시장 검증 필요(제휴처 확보 여부).
- **중개사 무료 6개월 종료 후 유료 전환율** — M4에서 가설 검증 전까지 수익 모델은 미확정.
