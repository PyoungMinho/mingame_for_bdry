## 0. 한 줄 요약 + 결정사항

**한 줄 요약**: 매일 쓴 일기 한 편을 AI가 4컷 만화로 변환해 SNS에 공유하고 1년 365편을 나만의 만화 아카이브로 쌓는 "습관형 만화 일기 SaaS".

**기획팀장 최종 결정 5선**
1. 컨셉 유지 — 글→4컷 만화, 다중 화풍, SNS 바이럴, "누구나 만화 작가" 슬로건 모두 유지.
2. **무료 정책 수정 (사장 결재 필요)** — 무료 "완성본 1개/일"은 경제성 붕괴 위험으로 보류, **저해상도+워터마크 미리보기 1회/일 + 유료 구독**으로 전환 제안.
3. 캐릭터 일관성은 LoRA 풀구현이 아닌 **"스타일라이즈 아바타 템플릿 8종 + 시드 고정"** 방식으로 MVP에서는 비용/일관성 동시 해결.
4. 화풍 명칭은 IP 분쟁 회피를 위해 **국가/스튜디오명 제거, 스타일 기술어**로 전면 교체 ("한국 웹툰체"→"감성 라인", "일본 만화체"→"대담한 펜선" 등).
5. 개발 착수 전 **Go/No-Go 게이트 3종(BEP 시뮬, 법무 검토, 내부 20명 테스트)** 의무 통과.

## 1. 서비스 개요 (Why / What / Who)

- **Why**: 일기는 쓰기 어렵고, 만화는 그리기 어렵다. 둘 다 "기록을 통한 자기 위안+공유 욕구"가 본질. AI로 진입장벽을 무너뜨려 "오늘을 콘텐츠로 남기는 습관"을 만든다.
- **What**: 텍스트 일기 입력 → AI가 4컷(발단/전개/절정/결말)으로 분해 → 선택한 화풍으로 만화 생성 → 캐릭터·대사 미세 조정 → SNS 공유 + 개인 아카이브 누적.
- **Who**: 10대 후반~30대 초반, 인스타·스레드·X에 일상을 가볍게 공유하고 싶지만 "글만으론 심심하고 사진은 부담스러운" 사용자.

---

## 2. 핵심 가치 제안 (KVP)

> **"오늘 하루를 한 줄만 적어도, 내 캐릭터가 4컷 만화로 살아난다. 1년 후엔 365편의 나만의 만화책."**

- 단순 도구가 아닌 **시리즈/아카이브**가 차별화 핵심.
- 경쟁사(투닝·AI Comic Factory): 1회성 도구. 우리: 매일 들어오게 만드는 **습관 SaaS**.

1. **수아 (24, 직장인 2년차)** — 인스타 스토리에 매일 뭐라도 올리고 싶지만 사진 찍기 귀찮음. 일기 앱은 3일 만에 그만둠. "글은 잘 쓰는데 그림은 못 그림".

**원가 가드레일**
- 4컷 1편 평균 생성 원가 목표: **$0.15 이하** (해상도/모델 최적화로 달성)
- BEP(손익분기): **유료 전환율 5% 기준 DAU 2,000 이내**에서 달성 가능해야 개발 진행
- 사용량 폭증 시 큐잉/지연 처리로 GPU 비용 상한 보호

> **사장 결재 필요 항목**: 사장 원안의 "무료 1편 완성본 제공"은 DAU 1,000만 되어도 월 $3,600~18,000 손실이 추정되어 SaaS 지속성 자체를 위협. "무료=미리보기"로 변경 동의 필요. 만약 무료 완성본을 고수한다면 **DAU 캡(예: 일 500편 선착순) + 광고 모델 동시 도입** 필요.

## 8. SNS 바이럴 전략

1. **공유 카드 표준화** — 1:1 정방형 + 4컷 슬라이드(인스타 캐러셀) + 세로 9:16(스레드/숏폼).
2. **워터마크 = 마케팅 자산** — 우측 하단 "툰일기" 로고 + QR 코드 (회원가입 랜딩). 무료 사용자는 제거 불가.
3. **해시태그 자동 삽입** — #툰일기 #오늘의툰 #AI일기 + 화풍별 태그.
4. **챌린지 캠페인** — "#30일툰일기" "#이번주하이라이트" 월간 운영.
5. **인플루언서 시드** — 출시 직전 일러스트/일상 크리에이터 20명에게 Pro 6개월 무상 제공.
6. **법무 게이트** — 원클릭 공유 기능은 변호사 검토 통과 후에만 활성화. MVP는 "저장 후 수동 공유" 우회.

## 10. Go/No-Go 검증 게이트 (개발 착수 전 통과 조건)

1. **경제성 시뮬레이션 통과** — DAU 2,000 가정 시 월 운영원가 ≤ 월 매출 (모델/해상도 PoC 기반).
2. **법무 검토 완료** — 외부 변호사 자문 받아 약관·개인정보처리방침·미성년자 플로우·AI 생성 고지 문구 확정.
3. **내부 알파 테스트** — 사내·지인 20명에게 1주일 사용 후 "이게 내 이야기 같다" 만족도 60% 이상, NPS ≥ +20.

세 가지 중 하나라도 미달이면 개발 STEP으로 넘어가지 않음.
