## 2. R2 양측 의견 종합 평가

### 2.1 창의팀 1순위 "유튜브 채널 분석기" 재검증

| 비평팀 가드레일 | 현 안에서의 충족도 | 판정 |
|---|---|---|
| (1) 데이터 소싱 법적 검토 | YouTube Data API v3는 공식 / quota 제한 존재 / TOS 준수 시 OK | △ 조건부 — quota 비용·캐싱 전략 필수 |
| (2) 수익 모델 디커플링 | 창의안은 광고 위주 → **재설계 필요** | X → ○ 로 수정 (어필리에이트+B2B API+뉴스레터) |
| (3) AI Overview 회피 (계산/비교/생성) | 채널 "분석 카드 생성"은 생성형이라 회피 가능, 단 "Social Blade 대안 검색"은 정보형 → 일부 잠식 | △ 조건부 — 생성/분석 카드 중심으로 포지셔닝 |
| (4) 글로벌 vs 한국 택일 | **글로벌 영문 우선** 권고 (Social Blade 트래픽의 80%가 글로벌, 한국 유튜버 시장은 좁음) | ○ 선택 완료 |
| (5) SEO 단독 의존 금지 | "내 채널 분석 카드 공유" 바이럴 훅 존재 → X(트위터)/Threads/유튜브 커뮤니티 탭 활용 | ○ |
| (6) Similarweb 사전 검증 | 진입 전 Social Blade/vidIQ/Tubebuddy 트래픽·키워드 갭 분석 필요 | △ 착수 전 1주 검증 필수 |

**결론**: 4개 ○ + 3개 △ — **조건부 합격**. 단, 다음을 수정:
- 수익 모델: 광고 의존 폐기 → **(a) vidIQ/Tubebuddy 어필리에이트, (b) MCN/마케팅 대행사 대상 B2B 임베드 API, (c) 크리에이터 뉴스레터 스폰서십** 3축으로 재설계.
- 콘텐츠 형태: 단순 통계 페이지 X → **"채널 진단 카드 + 성장 시뮬레이션 + 경쟁 채널 매치업"** 같은 생성/비교형 콘텐츠 중심.
- 사전 검증: 본 작업 착수 전 1주간 Similarweb·Ahrefs·구글 트렌드로 키워드 갭 검증.

### 2.2 비평팀 "숨은 카테고리"에서 추가 픽업

비평팀이 제시한 4개 중 **글로벌 시험·자격증 비교**가 가장 매력적:
- AI Overview 회피 강함 (개인 상황별 비교 = 생성형)
- 어필리에이트 친화 (시험 준비 코스, 교재)
- 글로벌 수요 크고 한국 한정 아님
- 경쟁이 의외로 노후 (Crunchprep, Magoosh blog 등)

→ **백업 B**로 채택.

### 2.3 탈락한 옵션

- **AI 모델 비교 대시보드**: 기술적으로 매력적이나 타겟이 좁고(개발자/PM), 5만 UV 도전적. → 백업 A로 유지 (B2B 가치 높음).
- **환율+해외송금 비교기**: Wise/Revolut 자체 비교 페이지가 압도적, 차별화 어려움. → 탈락.
